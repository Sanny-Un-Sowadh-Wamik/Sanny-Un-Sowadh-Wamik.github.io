# Sanny Un Sowadh Wamik - Portfolio

A modern, interactive portfolio website built with React, featuring a full admin dashboard for content management.

## Features

- **Interactive UI**: Smooth animations with Framer Motion
- **Particle Background**: Dynamic particle effects
- **Dark/Light Theme**: Toggle between themes with persistence
- **Command Palette**: Quick navigation with ⌘K / Ctrl+K
- **Admin Dashboard**: Full content management system
  - Profile management
  - Projects CRUD
  - Skills management
  - Education & Activities management
  - Data export/import
- **Data Visualizations**: Charts and statistics
- **Responsive Design**: Works on all devices
- **Contact Form**: Integrated with Web3Forms

## Tech Stack

- **React 18** - UI Framework
- **Vite** - Build tool
- **Tailwind CSS** - Styling
- **Framer Motion** - Animations
- **React Router** - Routing
- **Recharts** - Data visualization
- **React Icons** - Icon library
- **React Hot Toast** - Notifications

## Getting Started

### Prerequisites

- Node.js 18+
- npm or yarn

### Installation

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

### Admin Access

Navigate to `/admin` and use the following credentials:
- Username: `xxxx`
- Password: `xxxx`

## Project Structure

```
src/
├── admin/          # Admin dashboard pages
├── components/     # Reusable UI components
├── context/        # React context providers
├── data/           # Initial data
├── pages/          # Public portfolio pages
└── index.css       # Global styles
```

## Deployment

The site is automatically deployed to GitHub Pages using GitHub Actions when changes are pushed to the main branch.
